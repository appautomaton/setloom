# Lane Pack Contract

```text
track spec + human listening -> musical decisions
style pack                  -> lane routing + technical hygiene scaffolding
local corpus                -> evidence for future studies, not rules by itself
```

The harness may block or repair technical problems: clipping, unsafe low end,
missing mixable edges when the form asks for them, or broken file outputs. It
should not choose BPM, bass profile, kick pattern, groove identity, motif
behavior, or arrangement shape from pack defaults.

## Forbidden Pattern

```text
small corpus measurement -> global style rule -> silent default
```

Keep evidence, hypothesis, taste, and executable gates separate.
