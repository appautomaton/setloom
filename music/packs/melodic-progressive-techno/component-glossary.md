# Component Glossary

Shared names for listening notes. This file defines what to point at; it does
not assign musical ownership to GenAI, deterministic code, or a style pack.

| Component | What To Listen For |
| --- | --- |
| Kick | Pulse, weight, transient shape, relationship to bass |
| Bass | Motion, register, pressure, syncopation, restraint |
| Percussion | Top-loop motion, accents, near/far depth, phrase marking |
| Pad | Harmonic bed, air, warmth, width, emotional pressure |
| Lead | Foreground identity, contour, restraint, variation |
| Arp | Repetition, hypnotic motion, density, phrase evolution |
| Pluck | Short melodic/rhythmic punctuation |
| Stab | Chord hit, groove punctuation, tension cue |
| Drone | Sustained pressure or tonal floor |
| Atmosphere | Space, place, texture, continuity |
| Riser / uplifter | Tension increase into a boundary |
| Impact / downlifter | Arrival marker or energy reset |
| Vocals | Lead, texture, hook, breath, phrase, or atmosphere |

## Use

Write listening notes as:

```text
component + section + observation + requested direction
```

Example: `bass, drop 1: motion feels too square; try a less predictable second
phrase.` The note belongs to the track, not the style pack.
